### Premier Studios Demo Trial project

This project is a demo trial project for Premier Studios. It is a simple blog application that allows users to create posts and view all posts.

### Back-end

Routes available:

| Method | URL        | Description                                               |
|--------|------------|-----------------------------------------------------------|
